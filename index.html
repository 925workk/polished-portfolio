The only developer for a unique project created with Spacestation Gaming and Youtuber Shonduras. Quarter Machine uses randomly generated 3D assets for the creation of NFTs. Each week, a new 3D animated short film would be released on Youtube. If a holder’s NFT was featured in the specific episode, they would get paid out the Youtube royalties associated with that video for owning that NFT.
